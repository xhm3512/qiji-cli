## 自制前端脚手架,实现公司内部前端工程化
***

###   安装
#### mac安装
>sudo npm i qiji-cli -g


#### windows安装
>npm i qiji-cli -g
### 模块
#### 初始化
> qiji-cli init 目录名

创建完成后结构如下：
```
├─.gitignore
├─README.md
├─package-lock.json
├─package.json
├─command
|    └init.js
```
